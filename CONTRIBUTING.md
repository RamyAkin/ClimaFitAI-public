# Contributing to ClimaFitAI (Public Issue Tracker)

Thank you for your interest in improving ClimaFitAI! 🎉

This repository is the **public issue tracker** for ClimaFitAI. The source code is maintained privately.


## 💬 For Users & Contributors

If you'd like to report a bug, request a feature, or ask a question:

1. Go to the [**Issues**](https://github.com/RamyAkin/ClimaFitAI-public/issues) tab
2. Search to see if your issue has already been reported
3. Open a new issue using the appropriate template

Thank you for helping make ClimaFitAI better! 🙏
