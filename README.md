## 📋 About

This repository is the **public issue tracker** for ClimaFitAI.

The source code is kept private, but you are welcome to:

- 🐛 **Report bugs** you encounter
- ✨ **Request new features** or improvements
- 💬 **Ask questions** about the app

All feedback helps make ClimaFitAI better — thank you for taking the time!

## 🤝 How to Contribute

1. Go to the [**Issues**](https://github.com/RamyAkin/ClimaFitAI-public/issues) tab
2. Search to see if your issue has already been reported
3. Open a new issue using the appropriate template (**Bug Report** or **Feature Request**)

Please be as descriptive as possible so we can address your feedback quickly.
